# Changelog

All notable changes to Sogo Canvas will be documented in this file.

## 0.0.1

- Initial preview release of the Sogo Canvas custom editor for `.canvas` files.
- Added a theme-native infinite canvas with text, group, file, and image nodes.
- Added connector styling, background modes, snap-to-grid, and viewport persistence.
- Improved Marketplace packaging metadata and VSIX packaging behavior.
- Added safer custom editor handling for malformed documents and non-`file` workspaces.
